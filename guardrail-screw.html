สกรูและน็อตการ์ดเรล คุณภาพงานโครงการ

NTW Steel ผลิตและจำหน่ายสกรู โบลท์ และน็อตสำหรับราวกันอันตราย (guardrail) แบบครบชุด ทั้ง button head guardrail bolt, splice bolt, post bolt และ recessed guardrail nut ทุกชิ้นชุบสังกะสีแบบจุ่มร้อน (hot-dip galvanized) ตามข้อกำหนดงานถนนของกรมทางหลวง รองรับงานติดตั้งราวกันอันตราย W-beam และ Thrie-beam ในโครงการทางหลวง สะพาน และงานโครงสร้างพื้นฐานทั่วประเทศ

สกรูและน็อตการ์ดเรลของ NTW Steel ขึ้นรูปจากเหล็กกล้าคาร์บอนตามมาตรฐาน ASTM A307 Grade A (น็อตตาม ASTM A563 Grade A) พร้อมทางเลือกเกรดกำลังสูง 8.8 สำหรับจุดรับแรงพิเศษ หัวสกรูแบบ button/round head มาพร้อมคอกันหมุน (oval shoulder) ที่ล็อกเข้ากับร่องแผ่นราง ทำให้ติดตั้งได้เร็วโดยขันเพียงด้านเดียวและไม่ต้องจับหัวสกรู ทุกชิ้นผ่านการชุบกัลวาไนซ์แบบจุ่มร้อนบนฟาสเทนเนอร์ตาม ASTM A153 / AASHTO M232 Class C (เคลือบสังกะสีเฉลี่ยไม่น้อยกว่า 1.25 oz/ft² ≈ 381 g/m² หรือ ~53 ไมครอน) เพื่ออายุการใช้งานยาวนานในสภาพแวดล้อมริมทางและเขตชื้นเค็ม ทั้งนี้แผ่นราง (rail element) เองใช้เกณฑ์เคลือบ Class A ≥ 550 g/m² ตาม AASHTO M180 ซึ่งเป็นคนละมาตรฐานกับตัวสลัก พิกัดขนาด เกลียว และรัศมีหัวสกรูควบคุมให้ตรงแบบมาตรฐานกรมทางหลวง AASHTO M180 และรหัสชิ้นส่วนตาม AASHTO-AGC-ARTBA Guide to Standardized Highway Barrier Hardware พร้อมส่งมอบใบรับรองวัสดุ (mill certificate) และผลทดสอบแรงดึง เหมาะสำหรับผู้รับเหมางานถนน โรงงานผลิตราวกันอันตราย และงานโครงการปริมาณมากที่ต้องการทั้งของพร้อมสต๊อกและงานสั่งผลิตตามแบบ (OEM)

การใช้งาน

เหมาะกับงานแบบไหน

01

ราวกันอันตรายริมทางหลวง (Highway W-beam)

ยึดแผ่นราง W-beam เข้ากับเสา I-beam / C-post ตามแบบมาตรฐานกรมทางหลวง ด้วย post bolt 5/8"/M16 และ recessed nut

02

รอยต่อแผ่นราง (Rail splice joint)

splice bolt สั้น ~30–35 มม. ยึดแผ่นรางซ้อนทับต่อเนื่องกัน โดยทั่วไป 8 ตัวต่อรอยต่อ W-beam ให้แรงส่งผ่านต่อเนื่องเมื่อเกิดแรงปะทะ

03

จุดต่อ Thrie-beam และ transition

งาน containment สูง เช่น ทางโค้ง คอสะพาน จุดเปลี่ยนหน้าตัดราง ต้องการชุดสลักที่แข็งแรงและได้พิกัดตามแบบ

04

ราวกันอันตรายบนสะพาน (Bridge rail / parapet)

ยึดราวเหล็กเข้ากับโครงสร้างคอนกรีตหรือเหล็กขอบสะพาน พร้อมชุบกัลวาไนซ์ทนสภาพอากาศเปิดโล่ง

05

เกาะกลางถนนและทางแยก (Median barrier)

ระบบราวสองด้าน (double-sided) และจุดกลับรถ ต้องการสลักครบชุดในปริมาณมากต่อกิโลเมตร

06

งานเขตชายฝั่ง/ความชื้นสูง

เลือก HDG ความหนาเคลือบพิเศษหรือสแตนเลส SUS304/316 เพื่อยืดอายุการใช้งานในบรรยากาศเค็มชื้น

สกรูการ์ดเรลใช้เกรดและมาตรฐานอะไร?
guardrail bolt ผลิตจากเหล็กกล้าคาร์บอนตาม ASTM A307 Grade A (ความแข็งแรงดึง ≥ 415 MPa) ประกบน็อต ASTM A563 Grade A และชุบกัลวาไนซ์จุ่มร้อนตาม ASTM A153/AASHTO M232 Class C ตามข้อกำหนด AASHTO M180 และกรมทางหลวง มีเกรด 8.8 (≥ 800 MPa) ตามสั่งสำหรับจุดรับแรงสูง
การ์ดเรลใช้สกรูขนาดไหน M16 หรือ 5/8 นิ้ว?
สลักการ์ดเรลมาตรฐานคือ 5/8"-11 UNC (เส้นผ่านศูนย์กลาง 15.9 มม. ≈ M16) โดย splice bolt ยาวประมาณ 30–35 มม. และ post bolt ยาวตามชนิดเสา 5/8" กับ M16 ใกล้เคียงกันแต่เกลียวไม่สลับใช้ข้ามกัน จึงต้องสั่งสลักและน็อตเป็นชุดเดียวกัน
ชุบกัลวาไนซ์สกรูการ์ดเรลหนาเท่าไร ตรงงานถนนไหม?
ตัวสลักและน็อตชุบจุ่มร้อนตาม ASTM A153 / AASHTO M232 Class C เคลือบสังกะสีเฉลี่ยไม่น้อยกว่า 1.25 oz/ft² (~381 g/m² หรือ ~53 ไมครอน) ส่วนแผ่นรางเองใช้ Class A ≥ 550 g/m² ตาม AASHTO M180 ซึ่งเป็นคนละเกณฑ์กับตัวสลัก
splice bolt กับ post bolt ต่างกันอย่างไร?
splice bolt เป็นสลักสั้น (~30–35 มม.) ยึดรอยต่อแผ่นรางที่ซ้อนทับกัน โดยทั่วไปใช้ 8 ตัวต่อรอยต่อ W-beam ส่วน post bolt ยาวกว่า ใช้ยึดแผ่นรางเข้ากับเสาหรือบล็อกเอาต์ ปกติ 1 ตัวต่อเสา
post bolt สำหรับเสาไม้หรือบล็อกเอาต์ยาวเท่าไร?
post bolt สำหรับเสาไม้และบล็อกเอาต์ผลิตยาวพิเศษ 250–460 มม. (10–18 นิ้ว) ตามความหนาชุด block-out ส่วนการยึดเข้าเสาเหล็กมักใช้ยาวประมาณ 45–50 มม.
รองรับทั้ง W-beam และ Thrie-beam ไหม?
รองรับทั้งสองระบบ ใช้สลัก 5/8"/M16 เหมือนกัน แต่ Thrie-beam มีสามลอนและจุดต่อมากกว่า จึงใช้ splice bolt ต่อรอยต่อมากขึ้น เหมาะกับงาน containment สูง เช่น ทางโค้ง คอสะพาน และจุด transition
มีใบเซอร์ mill certificate และผลทดสอบแรงดึงให้ไหม?
มีครบ ทั้ง mill certificate ของวัตถุดิบ ผลทดสอบแรงดึง (tensile test) และใบรับรองความหนาการชุบกัลวาไนซ์ ออกตามล็อตการผลิตและเงื่อนไขผู้ควบคุมงานโครงการ
สั่งขั้นต่ำ (MOQ) เท่าไร และมีของพร้อมสต๊อกไหม?
ชุด 5/8"/M16 x 30/35/45 มม. พร้อม recessed nut มีสต๊อกพร้อมส่งสำหรับงาน W-beam มาตรฐาน สั่งได้ตั้งแต่หลักลัง ส่วนงานสั่งผลิตตามแบบ (OEM) MOQ เริ่มประมาณ 500–1,000 กก. ต่อขนาด
ระยะเวลาผลิตและจัดส่ง (lead time) นานแค่ไหน?
ของในสต๊อกจัดส่งภายใน 1–3 วันทำการทั่วประเทศ งานสั่งผลิตตามแบบพร้อมชุบกัลวาไนซ์จุ่มร้อนประมาณ 2–4 สัปดาห์ ขึ้นกับปริมาณและขนาด รองรับการทยอยส่งตามแผนติดตั้ง
การ์ดเรลชุบกัลวาไนซ์ใช้งานทนกี่ปี?
ในสภาพริมทางทั่วไป การชุบกัลวาไนซ์จุ่มร้อนตาม ASTM A153 ให้อายุกันสนิมได้หลายสิบปี ส่วนเขตชายฝั่งหรือความชื้นเค็มสูงแนะนำเพิ่มความหนาเคลือบหรือใช้สแตนเลส SUS304/316 เพื่อยืดอายุ

พร้อมสั่ง สกรูและน็อตการ์ดเรล แล้วหรือยัง?

ส่งแบบหรือสเปกมาให้เรา รับใบเสนอราคาราคาโรงงานพร้อมกำหนดส่งภายใน 24 ชั่วโมงทำการ
